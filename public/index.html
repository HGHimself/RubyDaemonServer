Deco CSS Framework

made by HG King

Making CSS Easier

Deco is a collection of CSS classes that facilitates the styling of your web pages. From everything to layout to aesthetic, Deco provides users with just about everything that they would need.

Aesthetic Features:

Deco

Deco draws inspiration from the 1920's Art Deco design style. The aesthetic emphasises bold geometric forms, bright colors, and exotic styles. To quote the linked wiki page: "Art Deco represented luxury, glamour, exuberance, and faith in social and technological progress." To tap into the art deco style, I made it a point to express these principles. A striking, glimmering gold border envelops every element. Colors like pearl, emerald, and ruby adorn the display giving off a luxurious appearance.

Drop Caps

The drop-cap class can be used on any paragraph tag. The result is gives the initial letter of the text a printing press styling. This works best when the paragraph is over three lines long when rendered in the page. A way to make sure this happens is to break the text into multiple columns, like in a newspaper layout. One should note, that due to the way CSS works, the font of this initial letter must be explicitly set in the stylesheet. It is recommended that the header/main font is used to add more emphasis.

Using the 'column' class makes it easy to divide a row up into columns.

You can use most fractions along with this class to designate a width percentage. Also remember to not use any styling classes on the same element that has a 'column-n' class. This will mess up the percentages calculation.
