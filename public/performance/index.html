Performance Measures

Wednesday September 25, 2019


 Hey all, if you have not read it yet, the server that your browser is communicating with is described in depth on the home page. I think it is very fascinating and I recommend checking it out.

 In the Functional Ruby Server implementation, I have something that logs a few numbers for every single connection made. Some of there are the client IP address, request time, response time, bytes sent, and request path. This log file can easily be turned into a JSON object which fits nicely into graphs. I made a couple of different graphs up so take the time to explore them. You may even see yourself in some of the numbers!
