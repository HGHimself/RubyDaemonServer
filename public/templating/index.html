Templating

Monday November 25, 2019

In any useful front end framework, there is some sort of templating system that will make life easier for the developer who. This developer wants, as any good developer should, to build many small components that will eventually combine in to large complex components. When working on this framework, I had found myself copying and pasting a ton of different bits and pieces into files and changing every single instance when I decided I wanted something to look differenly. In order to solve this problem, I think it is time for me to finally set out to build a system that will make this templating happen and in turn take a lot of boring work out of this site. Here is how I will go about designing this templator.

Regex Fu

To start off, I think it is very important to first figure out how we are going to find the areas inside our file that need to be templated. It seems obvious to me that we are going to have to use some sort of special pattern and to find this pattern, we will need a regular expression.
