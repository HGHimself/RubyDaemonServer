Loading Slug

Wednesday September 18, 2019

 Hello, check out this sweet animation that I made using pure Javascript, sorting algorithms, and sweet math. I call it the Loading Slug because it was originally made for a loading animation in a project of mine. The slug part can be figured out with a little imagination.

 The sorts are nice to look at but I like to mess around with the color offset and multiplier. The method I use to set the colors of the bars in a spectrum is a custom function I wrote up a long time ago. Baiscally it uses three Simple Harmonic Motion functions to make each Red, Green, and Blue color values go from 0 - 255 at the proper times in respect to eachother in order to hit all of the primary colors. Any Multiplier that is close to pi or a multiple of pi will do some funky stuff (cus circles ya' know). Check out the colors tab at the top for more color fun.
