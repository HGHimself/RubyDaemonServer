Wiki Color Scraper

Created By HG King

This webapp will scrape all the color names, hex codes, and wiki page links from the three pages found:

  • List_of_colors:_A–F
  • List_of_colors:_G–M
  • List_of_colors:_N-Z

To use, copy and paste the text from one of the above links into the input field below. The script will go forth and scrape the code from the site and then insert it into the databse. From there the script will print out every entry in the database so far. Any false or empty imput will result in the display of the database.
